# 💻 Computer Usage Survey Form
 
A simple, browser-based survey form built with pure HTML and CSS — no frameworks or JavaScript required. It collects information about users' computer usage habits as part of a statistical study.
 
---
 
## 📋 Overview
 
This project is a static HTML survey form designed to gather data on computer usage patterns. Responses are anonymous and intended for statistical analysis only.
 
---
 
## 🗂️ Project Structure
 
```
survey-form/
└── survey_form.html    # Main survey form (single-file project)
```
 
---
 
## ✨ Features
 
- **Personal Details** — Name, email address, and age fields
- **Skill Level** — Dropdown to self-assess computer proficiency (No knowledge → Expert)
- **Favorite OS** — Radio buttons for GNU/Linux, macOS, Windows, FreeBSD
- **Hardware Owned** — Checkboxes for Desktop, Laptop, Mini-PC, Server, Single Board Computer
- **Additional Comments** — Free-text textarea for extra details
- **Form Validation** — Uses HTML5 `required` attributes for basic client-side validation
---
 
## 🚀 Getting Started
 
No installation or build step needed. Just open the file in any modern browser.
 
```bash
# Clone the repository
git clone https://github.com/your-username/survey-form.git
 
# Open the form
cd survey-form
open survey_form.html       # macOS
xdg-open survey_form.html   # Linux
start survey_form.html      # Windows
```
